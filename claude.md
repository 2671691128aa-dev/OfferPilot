# OfferPilot MVP 产品需求文档（PRD）

版本：V1.0

项目类型：AI 求职辅助 Web 应用

开发模式：Vibe Coding + AI辅助开发

目标：打造一个帮助大学生生成实习简历、优化求职材料的 AI 工具。


---

# 第一章 项目概述（Project Overview）


# 1.1 项目背景


随着互联网行业竞争不断增加，越来越多大学生在寻找实习机会时遇到以下问题：

- 不知道如何制作专业简历；
- 不知道如何描述自己的项目经历；
- 不清楚目标岗位需要哪些技能；
- 简历内容无法匹配企业招聘需求；
- 缺少针对性的求职反馈。


尤其对于计算机、设计、数字媒体等专业学生来说，很多学生拥有课程项目或个人项目，但由于缺少求职经验，无法将这些经历转化为符合企业招聘标准的简历内容。


因此，OfferPilot 希望利用人工智能技术，帮助大学生快速完成：

```
个人经历整理

↓

AI分析

↓

生成岗位匹配简历

↓

优化求职材料

↓

提升实习竞争力
```


---

# 1.2 产品定位


## 产品名称

OfferPilot


## 产品定位

OfferPilot 是一个基于大语言模型（LLM）的 AI 实习求职助手。


核心目标：

帮助大学生利用 AI 快速生成更加专业、符合岗位要求的实习简历。


产品关键词：

```
AI

Resume

Career

Internship

Student
```


---

# 1.3 产品目标


## MVP阶段目标


在第一版中，实现以下核心能力：


### 1. AI生成简历

用户输入：

- 基础信息；
- 技能；
- 项目经历；
- 目标岗位。


AI自动生成：

- 个人简介；
- 技能描述；
- 项目经历优化；
- 简历结构。


---

### 2. AI优化简历


用户上传或输入已有简历。


AI分析：

- 内容问题；
- 表达问题；
- 岗位匹配度。


并提供：

- 修改建议；
- 优化后的描述。


---

### 3. JD岗位分析


用户输入招聘岗位描述。


AI分析：

- 岗位需要技能；
- 用户匹配程度；
- 当前不足。


---

### 4. PDF简历导出


用户可以：

- 选择简历模板；
- 生成PDF；
- 下载保存。
